## Description
A minimal Spring Boot web application demonstrating:
- a plain-text controller response using `@ResponseBody`
- an MVC controller that renders a Thymeleaf template
- serving a static image

## Requirements
- Java (install a JDK, e.g. Java 17 or newer)
- Maven (or use the included `mvnw` wrapper)
- IDE (optional) such as IntelliJ IDEA

## How to run
1. Run the application using your IDE or with `mvn spring-boot:run`
2. Open a browser and visit:
    - http://localhost:8080/greeting?name=YourName
